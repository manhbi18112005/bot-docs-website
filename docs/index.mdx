## Supporting the bot

We basically need money to continuously host the bot on costly servers.

There are several ways you can support us:

* Supporting us through [Patreon](https://www.patreon.com/manhbi18112005)

## Reporting issues

If you're having issues with the bot, check out the [Frequent Searches](others/frequent-searches.mdx) page. If you do not find a solution there, you can also join the [support server](https://bot.nonamestudio.xyz/discord) and ask for help there.

## Making suggestions

If you have a suggestion for a new feature or improvements, you can either make a feedback at [this link](../feedback) or open an issue in the [GitHub](https://github.com/manhbi18112005/bot-docs-website) for the bot.
